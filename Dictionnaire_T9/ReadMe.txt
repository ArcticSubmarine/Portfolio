Fichiers fournis :
	- fichier .h : main.h
	- fichiers .c : main.c
	- fichiers .txt : dictionnaire.txt, mots_courants.txt, mots_courants_optimized.txt, ReadMe.txt



Modalités d'utilisation :

Executer le fichier main.c : le menu se lance automatiquement. Sélectionner l'une des options proposées en 
saisissant le chiffre correspondant.

*Rechercher un mot : saisir "1" puis entrer les premières lettres du mot recherché ; trois propositions vont 
alors s'afficher sur la console. Possibilité d'effectuer une autre recherche en choisissant de "continuer" 
saisir "oui").
*Ajouter un mot :  saisir "2" puis entrer le mot à ajouter au dictionnaire des mots courants.
*Supprimer un mot : saisir "3" puis entrer le mot à supprimer du dictionnaire des mots courants.
*Quitter : saisir "4". Quitte le programme de gestion du dictionnaire des mots courants.
